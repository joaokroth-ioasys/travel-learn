# Journey: City Life (Deep Dive)

## Concept

A journey built around life **inside a city**. It is intentionally generic, then
varies by language/country so the vocabulary fits the local context.

- A generic city, no fixed map.
- Multiple **locations** within that city.
- The learner selects a location and learns vocabulary specific to it.

## Goal

Capture the main situations a **tourist** — or someone moving to the country —
will realistically go through. For each situation, build a focused vocabulary
set so the learner is prepared for that real-world journey.

## Example locations

- Airport
- Train station
- Snack bar / café
- Gym
- Restaurant
- Public services
- Doctor / hospital
- Tourist guide

## Naming

Working names: **Deep Dive** or **City Life**.
